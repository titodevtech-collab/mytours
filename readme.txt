== Description ==

MyTours is a simple transportation booking plugin. It allows you to create tours and accept reservations via a frontend form.

== Installation ==

1. Upload the `mytours` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Create some Tours in the 'Tours' menu.
4. Use the shortcode `[mytours_booking]` on a page to display the booking form.
